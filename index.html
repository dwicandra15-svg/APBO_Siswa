💬 Pengaturan Diskusi Kelas

Fitur diskusi memerlukan tabel chat_apbo di Supabase dengan kebijakan public access agar dapat dibaca oleh APBO Siswa.

📋 SQL Setup Tabel Chat (jalankan sekali di Supabase SQL Editor)
🔧 SQL Perbaiki Kolom — Jalankan di Supabase SQL Editor jika ada error kelas_nama
📌 Cara Kerja Sinkronisasi
  • Guru kirim pesan → simpan ke chat_apbo
  • APBO Siswa baca berdasarkan kelas_nama
  • Realtime via Supabase Realtime
  • RLS: semua user bisa baca/tulis (public)
✅ Syarat Sinkronisasi
  • URL & Key Supabase sama di kedua aplikasi
  • Nama kelas di APBO Siswa sama persis
  • Tabel chat_apbo sudah dibuat
  • Realtime diaktifkan di Supabase
